# Module 2: Thematic Vocabulary for CELGA 2
## Vocabulario por temas (B1)

---

## Overview

Module 1 covered the core, high-frequency words. This module groups vocabulary by the **everyday topics CELGA 2 reading, writing and speaking tasks return to again and again**: daily routine, work, health, travel, environment, technology and feelings. Each list stays at B1 — words you should be able to **use**, not just recognize.

**How to use it:** one theme at a time; put the words into a flashcard deck with the example phrase; before a writing or speaking task on a topic, skim the matching theme and aim to use 5+ items.
