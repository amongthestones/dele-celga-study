# Module 3: False Friends and Common Errors
## Falsos amigos y errores frecuentes (DELE B2)

---

## Overview

At B2, the errors that cost you marks are rarely "big" grammar mistakes — they're **false friends**, **wrong prepositions**, and **collocations that don't transfer** from English. This module gathers the ones that show up most in writing and speaking.

**How to use it:** read for recognition, then keep a personal short-list of the 10–15 that *you* actually get wrong, and drill those.

---

## Part 1: False Friends (English ↔ Spanish)

| Spanish word | Actually means | English false friend | English word is really… |
|--------------|----------------|----------------------|--------------------------|
| actual | current, present-day | "actual" | real, verdadero |
| actualmente | currently, nowadays | "actually" | en realidad, de hecho |
| realizar | to carry out | "realize" | darse cuenta de |
| sensible | sensitive | "sensible" | sensato, razonable |
| asistir a | to attend | "assist" | ayudar |
| atender | to attend to, serve | "attend" | asistir a |
| éxito | success | "exit" | salida |
| salir | to go out / leave | "save" | guardar, ahorrar |
| soportar | to put up with, bear | "support" | apoyar, mantener |
| pretender | to intend, aim, claim | "pretend" | fingir, simular |
| embarazada | pregnant | "embarrassed" | avergonzado/a |
| introducir | to insert | "introduce (a person)" | presentar |
| molestar | to bother, annoy | "molest" | abusar (de), acosar |
| constipado | having a cold | "constipated" | estreñido |
| largo | long | "large" | grande |
| recordar | to remember/remind | "record" | grabar |
| compromiso | commitment, engagement | "compromise" | acuerdo, término medio |
| discutir | to argue | "discuss" | hablar de, tratar |
| en absoluto | not at all | "absolutely" | desde luego, totalmente |
| sensato | sensible | — | — |
| carpeta | folder | "carpet" | alfombra |
| ropa | clothes | "rope" | cuerda |
| sopa | soup | "soap" | jabón |
| una vez | once (one time) | "a vez" | — |

> Note `pretender`: *Pretendo aprobar el examen* = "I aim/intend to pass" — not "I pretend to". And `discutir` leans toward "argue/quarrel", while neutral "discuss" is *hablar de* or *tratar*.

## Part 2: Preposition Errors (the silent mark-losers)

### Verbs whose preposition differs from English

| Spanish | English | Note |
|---------|---------|------|
| depender **de** | depend **on** | not *en* |
| consistir **en** | consist **of** | not *de* |
| soñar **con** | dream **of/about** | |
| pensar **en** | think **about** | *pensar de* = have an opinion of |
| confiar **en** | trust / rely **on** | |
| casarse **con** | marry / get married **to** | |
| enamorarse **de** | fall in love **with** | |
| acordarse **de** | remember | reflexive + de |
| darse cuenta **de** | realize | |
| tardar **en** | take (time) **to** | *Tardé en entenderlo* |
| quejarse **de** | complain **about** | |
| contar **con** | count **on** / have | |

### Por vs Para (the classic B2 distinction)

| POR (cause, means, exchange, duration, "through") | PARA (purpose, destination, deadline, recipient) |
|---|---|
| Lo hago **por** ti (for your sake) | Es **para** ti (for you, recipient) |
| **por** la mañana (during) | **para** mañana (by tomorrow) |
| pasar **por** el parque (through) | salir **para** Madrid (towards) |
| **por** 50 euros (in exchange) | estudiar **para** médico (goal) |
| **por** correo (by means of) | **para** mí (in my opinion) |

## Part 3: Ser / Estar Errors

- **Location of an event** → **ser**: *La reunión **es** en la sala 5.* (not *está*)
- **Location of a thing/person** → **estar**: *El libro **está** en la mesa.*
- **Adjectives that change meaning** (see also Grammar Module 2): *es listo* (clever) vs *está listo* (ready); *es rico* (rich) vs *está rico* (tasty).

## Part 4: Grammar/Usage Errors Common at B2

| ❌ Error | ✓ Correct | Why |
|----------|-----------|-----|
| Si **tendría** dinero… | Si **tuviera** dinero… | no conditional after *si* |
| Espero que **vienes** | Espero que **vengas** | subjunctive after *esperar que* |
| **Hace** dos años **que** vivo aquí ✓ / *Vivo aquí desde hace dos años* ✓ | both fine | learn both patterns |
| La gente **son** | La gente **es** | *gente* is singular |
| **Hay** mucho**s** problema**s** ✓ / *Hay mucho problemas* ❌ | agreement | |
| **Me gusta** los libros | **Me gustan** los libros | verb agrees with the thing liked |
| Es **el** mejor **que** he visto | + subjunctive sometimes: *que haya visto* | superlative + relative |
| **Realizar** que… | **Darse cuenta de** que… | false friend |
| Pienso **de** ir | Pienso **en**/Ø ir | *pensar* + infinitive, no prep |

## Part 5: Latin America vs Spain (recognize both)

- **vosotros** (Spain) vs **ustedes** for informal plural (Latin America)
- **vos** (Argentina, parts of Central America): *vos tenés, vos sos*
- Present perfect: Spain *hoy he comido* / much of LatAm *hoy comí*
- Vocabulary: *coche* (Spain) / *carro* (much of LatAm); *ordenador* / *computadora*; *móvil* / *celular*

The DELE accepts any standard variety consistently — just don't mix *vosotros* and *ustedes* within one text.

## Exercises

### Exercise 1: Choose the right word
1. (Actualmente / En realidad) no me gusta, aunque parezca lo contrario.
2. Voy a (asistir / atender) a la conferencia.
3. No puedo (soportar / apoyar) el ruido.
4. Me (di cuenta / realicé) de mi error.
5. Está (embarazada / avergonzada): va a tener un bebé.

### Exercise 2: Preposition
1. Todo depende ___ ti.
2. El curso consiste ___ diez sesiones.
3. Sueño ___ viajar a Japón.
4. Tardé una hora ___ llegar.
5. Me quejé ___ el servicio.

### Exercise 3: Fix the error
1. Si tendría tiempo, iría.
2. Espero que vienes mañana.
3. Me gusta los deportes.
4. La gente son egoístas a veces.
5. La fiesta está en mi casa.

## Answer Key

### Exercise 1:
1. En realidad  2. asistir  3. soportar  4. di cuenta  5. embarazada

### Exercise 2:
1. de  2. en  3. con  4. en  5. de

### Exercise 3:
1. Si **tuviera** tiempo, iría.
2. Espero que **vengas** mañana.
3. Me **gustan** los deportes.
4. La gente **es** egoísta a veces.
5. La fiesta **es** en mi casa.
