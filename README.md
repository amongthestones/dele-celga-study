# Language Exam Study Guide
## DELE B2 (Spanish) + CELGA 2 (Galician)

A study guide covering grammar, vocabulary, and exam-specific strategy for two language certifications:

- **DELE B2** — Diploma de Español como Lengua Extranjera, upper-intermediate Spanish (Instituto Cervantes).
- **CELGA 2** — Certificado de Lingua Galega, B1 Galician (Xunta de Galicia).

These are study materials, not a schedule. Work through modules at whatever pace fits. Each file stands on its own.

**Exam format:** 4 skill groups.
- Reading: 70 min, 4 tasks, 36 items
- Listening: 40 min, 5 tasks, 30 items
- Writing: 80 min, 2 tasks, 150–180 words each
- Speaking: ~15 min + 15 min prep, 3 tasks

## Suggested Order

If starting fresh, most learners benefit from:

1. **Diagnostic test** (`07_Exam_Prep/DIAGNOSTIC_TEST.md`) — identify weak areas.
2. **Grammar modules** in order — these build on each other.
3. **Vocabulary modules** alongside grammar, not after.
4. **Skill-specific modules** (reading, writing, listening, speaking) once grammar basics are solid.
5. **Worksheets and past papers** in the weeks before the exam.

Cheatsheets are reference material — use them for quick review, not initial learning.
