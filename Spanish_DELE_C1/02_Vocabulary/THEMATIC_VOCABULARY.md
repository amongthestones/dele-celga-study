# Thematic Vocabulary — DELE C1
## 10 Themes | ~35 words each | Study one theme per Session 2

**How to use:**
1. In Session 2, find the next theme you haven't studied
2. Read through all words once
3. Add every word to Anki (front: Spanish word, back: meaning + example)
4. Mark the theme with today's date when done
5. When writing an essay on a related topic, open this file first

**Register key:** F = formal/written | N = neutral | C1 = distinctly C1 level (not basic B2)

## Bonus: Phrases Useful for Writing (All Themes)

These are not vocabulary items — they are C1-level writing connectors and constructions. Use them in your essays.

| Function | Phrases |
|----------|---------|
| **Introducing a complex argument** | Cabe preguntarse si... / Conviene matizar que... / No es casual que... |
| **Conceding a point** | Si bien es cierto que..., no es menos verdad que... / Aunque se podría argumentar que..., lo cierto es que... |
| **Adding nuance** | Ello no obsta para que... / Dicho esto, conviene señalar que... / Más aún, ... |
| **Citing evidence** | A tenor de los datos disponibles... / Según se desprende de... / Los estudios revelan de forma consistente que... |
| **Drawing conclusions** | En última instancia, ... / Todo ello apunta a que... / De lo expuesto se concluye que... |
| **Presenting contrast** | Frente a esta postura, cabe oponer... / A diferencia de lo que suele sostenerse... / Lejos de ser un problema aislado, ... |
| **Formal qualification** | En términos generales... / En líneas generales... / Salvo excepciones... |
| **Calling for action** | Resulta imprescindible que... / Se impone la necesidad de... / No cabe sino exigir que... |
