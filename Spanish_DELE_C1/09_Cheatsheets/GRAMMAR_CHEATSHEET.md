# Spanish Grammar Cheatsheet for DELE C1
## Quick Reference Guide

## SUBJUNCTIVE TRIGGERS

### Always Subjunctive
| Trigger | Example |
|---------|---------|
| antes de que | Antes de que **llegue** |
| para que | Para que **entienda** |
| a menos que | A menos que **llueva** |
| sin que | Sin que lo **sepa** |
| con tal de que | Con tal de que **vengas** |
| ojalá | Ojalá **estuviera** aquí |
| como si | Como si **supiera** todo |

### Subjunctive with These Verbs
- querer que, desear que, esperar que
- pedir que, rogar que, sugerir que
- mandar que, ordenar que, prohibir que
- molestar que, gustar que, sorprender que
- dudar que, negar que, no creer que

### Indicative (Certainty)
- creer que, pensar que
- es verdad que, es cierto que
- está claro que, es evidente que
- no dudar que, no negar que

## CONDITIONAL SENTENCES (SI CLAUSES)

| Type | Si clause | Main clause | Example |
|------|-----------|-------------|---------|
| 1 (Real) | Present | Future/Present/Imperative | Si llueve, no iré |
| 2 (Hypothetical) | Impf. Subj. | Conditional | Si tuviera, viajaría |
| 3 (Impossible past) | Pluperf. Subj. | Cond. Perfect | Si hubiera sabido, habría ido |

**Never:** Si + conditional, si + future

## TENSE QUICK REFERENCE

### Indicative
| Tense | Example (hablar) | Use |
|-------|-----------------|-----|
| Present | hablo | Now, habits, facts |
| Preterite | hablé | Completed past |
| Imperfect | hablaba | Ongoing past, habits |
| Future | hablaré | Future, probability now |
| Conditional | hablaría | Hypothetical, probability past |
| Pres. Perfect | he hablado | Recent past, experience |
| Pluperfect | había hablado | Past before past |

### Subjunctive
| Tense | Example | Use |
|-------|---------|-----|
| Present | hable | Present/future subjunctive |
| Imperfect | hablara/hablase | Past subj., hypothetical |
| Pres. Perfect | haya hablado | Recent past subjunctive |
| Pluperfect | hubiera hablado | Past of past subj. |

## SER vs ESTAR

### Use SER for:
- Identity: Soy profesor
- Origin: Soy de México
- Time: Son las tres
- Material: Es de madera
- Possession: Es mío
- Events: La fiesta es aquí
- Passive voice: Fue construido

### Use ESTAR for:
- Location: Está en casa
- State/condition: Está cansado
- Progressive: Está trabajando
- Result: Está roto
- Temporary: Está enfermo

### Adjectives that Change Meaning
| With SER | With ESTAR |
|----------|------------|
| listo = clever | listo = ready |
| malo = bad | malo = sick |
| verde = green | verde = unripe |
| aburrido = boring | aburrido = bored |
| rico = rich | rico = delicious |

## KEY CONNECTORS

### Addition
además, asimismo, también, incluso, es más

### Contrast
sin embargo, no obstante, aunque, a pesar de

### Cause
porque, ya que, puesto que, debido a

### Consequence
por lo tanto, por consiguiente, así que, de ahí que

### Conclusion
en conclusión, en definitiva, por último

### Opinion
en mi opinión, a mi juicio, desde mi punto de vista

## PREPOSITIONS

### POR vs PARA
| POR | PARA |
|-----|------|
| Cause: por miedo | Purpose: para aprender |
| Duration: por dos horas | Deadline: para mañana |
| Through: por la ciudad | Destination: para Madrid |
| Exchange: por 10 euros | Recipient: para ti |
| Per: 100 km por hora | Opinion: para mí |
| Agent: escrito por | Comparison: para un niño |

### Common Verb + Preposition
- pensar en, soñar con, depender de
- acordarse de, olvidarse de
- consistir en, confiar en
- alegrarse de, quejarse de

## VERBAL PERIPHRASES

### Obligation
- tener que + inf (have to)
- deber + inf (should)
- hay que + inf (one must)

### Progressive
- estar + gerund (is doing)
- seguir + gerund (continue doing)
- llevar + time + gerund (have been doing for)

### Beginning
- empezar a + inf
- ponerse a + inf (suddenly start)
- echarse a + inf (burst into)

### Ending
- dejar de + inf (stop)
- acabar de + inf (have just)
- terminar de + inf (finish)

## RELATIVE CLAUSES

### Relative Pronouns
- que (that, which, who) - most common
- quien/quienes (who) - after preposition for people
- el/la/los/las que (the one(s) that)
- el/la cual, los/las cuales (formal)
- cuyo/a/os/as (whose)
- donde (where)
- lo que (what, that which)

### With Prepositions
- el libro **del que** hablo (about which)
- la persona **con quien** trabajo (with whom)
- la ciudad **en la que** vivo (in which)

## QUICK VERB IRREGULAR STEMS

### Present Subjunctive (from yo present)
- tengo → tenga
- vengo → venga
- digo → diga
- hago → haga
- pongo → ponga
- salgo → salga

### Imperfect Subjunctive (from 3rd plural preterite)
- tuvieron → tuviera
- vinieron → viniera
- dijeron → dijera
- hicieron → hiciera
- fueron → fuera

### Future/Conditional (irregular stems)
- tener → tendr-
- poner → pondr-
- venir → vendr-
- salir → saldr-
- poder → podr-
- saber → sabr-
- querer → querr-
- hacer → har-
- decir → dir-

## COMMON ERRORS CHECKLIST

- [ ] No conditional after "si" (Si tuviera, NOT si tendría)
- [ ] Como si + imperfect/pluperfect subjunctive
- [ ] Antes de que + subjunctive (always)
- [ ] Ser for events, estar for location of things
- [ ] Agree adjectives with nouns
- [ ] Use subjunctive for unknown antecedent
- [ ] Distinguish deber (obligation) from deber de (probability)
