# DELE C1 — Session Cards
## Open this file. Find your session. Do it. Close it.

No decisions. No planning. Work through the 5-session cycle in order, then repeat.
Track which session you're on in `essays_log.md` or a sticky note.

---

## The Cycle

| Session | Skill | Duration |
|---------|-------|----------|
| 1 — Writing A | New essay draft | 40 min |
| 2 — Vocabulary | Thematic study + Anki | 40 min |
| 3 — Listening | Active practice + notes | 40 min |
| 4 — Writing B | Model analysis + self-review | 40 min |
| 5 — Reading | Timed article or exam task | 40 min |
| Saturday | Exam practice section | 60 min |

Do sessions 1–5 on weekdays. Saturday is separate.

---

## Session 1 — Writing A: New Essay Draft

**Duration:** 40 min  
**What you need:** `ESSAY_PROMPT_BANK.md`, timer, blank document

**Steps:**
1. Open `ESSAY_PROMPT_BANK.md`
2. Find the next prompt you haven't written yet (work through in number order — don't pick, just do the next one)
3. Set a timer for **35 minutes**
4. Write your response. Stick to the word count in the prompt. Don't stop to look things up.
5. When the timer ends, stop writing
6. Save as: `04_Writing/drafts/essay_[number]_[date].md`
7. Note the prompt number in your essays log

**Done when:** File saved, timer stopped.

---

## Session 2 — Vocabulary: Thematic Study

**Duration:** 40 min  
**What you need:** `THEMATIC_VOCABULARY.md`, Anki (app or web)

**Steps:**
1. Open `THEMATIC_VOCABULARY.md`
2. Find the next theme you haven't studied (work through in order)
3. Read through all words in the theme once without adding them yet (5 min)
4. Add words to Anki: front = Spanish word, back = English meaning + the example sentence from the list (20 min)
5. Do one Anki review session — new cards only (10 min)
6. Mark the theme as done in the vocabulary file with today's date

**Done when:** All words added to Anki, review session complete, theme marked.

> **Note for essays:** When you write your next essay on a related topic, open `THEMATIC_VOCABULARY.md` for that theme first and try to use 5+ words from it.

## Session 3 — Listening: Active Practice

**Duration:** 40 min  
**What you need:** Headphones + one resource from the rotation below

**Resource rotation** (go through a → b → c → d → e, then repeat):

| Letter | Resource | Where |
|--------|----------|-------|
| a | Hora 25 (Cadena SER) | Podcast app / cadenaser.com |
| b | La Ventana (Cadena SER) | Podcast app / cadenaser.com |
| c | El País Semanal Podcast | Podcast app / elpais.com |
| d | TED en Español | ted.com/talks — filter: Spanish |
| e | Documentos RNE | Podcast app / rtve.es |

**Steps:**
1. Find your current letter in the rotation (a, b, c, d, or e)
2. Pick any episode and listen for **20–25 minutes**. While listening, take brief notes: topic, 3 key points, any words you don't know
3. Without re-listening: write **4–6 sentences in Spanish** summarizing the main argument (10 min)
4. Look up the words you noted. Add the **5 most useful** to Anki (5 min)
5. Move to the next letter in your rotation

**Done when:** Summary written, 5 words added to Anki.

## Session 4 — Writing B: Self-Review + Model Analysis

**Duration:** 40 min  
**What you need:** Your most recent saved essay draft, `04_Writing/MODULE_01_Writing_Guide.md`

**Steps:**
1. Open your most recent essay draft
2. Run the DELE rubric check (15 min):
   - **Task completion:** Did you address every point the prompt asked for? Mark any you missed.
   - **Coherence:** Does each paragraph have one clear main idea? Are your connectors logical?
   - **Linguistic range:** Circle any word you used more than once. Rewrite with a synonym.
   - **Accuracy:** Underline any grammar you're unsure about. Check it.
3. Rewrite **one paragraph only** — the weakest one (15 min)
4. If you have *El Cronómetro C1*: read one model answer for the same text type. Write down **3 phrases or structures you want to use in your next essay** (10 min)

**Done when:** Rubric check complete, one paragraph rewritten, 3 phrases noted.

---

## Session 5 — Reading: Timed Practice

**Duration:** 40 min

### Option A — If you have *El Cronómetro C1* (use this once you have it)

1. Open to the next reading task in sequence
2. Set the timer for the official time listed in the book
3. Complete the task. No pausing, no dictionary.
4. Mark your answers against the answer key
5. Review every wrong answer: write one sentence explaining why you got it wrong

**Done when:** Task completed, errors explained.

---

### Option B — Before *El Cronómetro C1* arrives

1. Go to **elpais.com** → Opinión → pick any *columna* (column) or editorial (600+ words)
2. Set timer: **12 minutes** to read it once. No stopping.
3. Without re-reading: write **5 bullet points** summarizing the argument (5 min)
4. Re-read the article and check your summary. What did you miss? Write a note. (8 min)
5. Find **5 words or expressions** you didn't know or wouldn't have written yourself. Add to Anki. (10 min)

**Done when:** Summary written, gap noted, 5 words in Anki.

---

## Saturday — Exam Practice

**Duration:** 60 min  
**What you need:** *El Cronómetro C1*

**Steps:**
1. Choose one complete exam section: Reading, Listening, or Writing (rotate each week)
2. Set the official exam timer for that section
3. Complete it. No pausing, no re-reading instructions, no dictionary.
4. Mark your work using the answer key
5. Calculate your score as a percentage
6. Review wrong answers — what was the pattern? Note it.
7. Log your score in the progress table below

**Done when:** Section scored, pattern identified, score logged.
